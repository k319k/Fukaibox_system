1. デザインの物理法則 (Physical Constants)

AIエージェントは以下の数値を「不変の定数」とし、これ以外の数値を生成してはならない。

A. 8dp グリッド・スペーシング (Spacing System)

2px (XXXS): ボーダー微調整用。

4px (XXS): gap-1（チップ/ボタン内アイコン間隔）。

8px (XS): gap-2（要素間隔）。

12px (Small): p-3（サブリスト）。

16px (Medium): p-4, gap-4（標準パディング/フォーム行間）。

24px (Large): p-6, gap-6（コンポーネント間ギャップ）。

32px (XL): p-8, gap-8（メインボード内部）。

48px (XXL): gap-12（セクション区切り）。

64px (XXXL): p-16（ページ上下）。

B. 角丸階層 (Radius Hierarchy)

Extra Large (28px): rounded-[28px] (Main Card, Modal, Drawer, Main Container)

Large (20px): rounded-[20px] (Sidebar, Table, Accordion, Image, Secondary Card)

Medium (16px): rounded-[16px] (Input, Alert, User, Avatar, Skeleton)

Pill (Full): rounded-full (Button, Chip, Tab, Switch, Search Bar)

2. レイアウト構造 & サイズ決定 (Architecture & Sizing)

A. 決定論的サイズ指定 (Deterministic Sizing)

「文字数によってカードやコンテナの幅が変わる」実装は厳禁とする。

原則: コンテナ（特にCard）には必ず w-full または固定グリッド（例：col-span-4）を指定せよ。

例外: w-fit は小さなポップオーバーやチップのみに許容する。

指示: コンテンツ量に関わらず、レイアウトのグリッドに沿った一定の幅を維持することで、画面の整列（Alignment）を死守せよ。

B. 背景とサイドバーの分離

Main Canvas: 必ず #fff8f6 (bg-background)。

Sidebar: 背景は #fceae7 (bg-content1)。rounded-r-[28px] または rounded-[28px] (Floating) を適用。

3. コンポーネント完全網羅定義 (The Encyclopedia)

AIは以下のPropsと classNames を強制的に適用せよ。

3.1 ナビゲーション & アクション

Accordion: variant="light" item: "bg-content1 rounded-[20px] px-6 mb-2"

Breadcrumbs: separator: "text-default-300", 最終要素は font-bold text-primary

Button: radius="full" font-bold active:scale-95 transition-all

重要: アイコン整列: className="flex items-center justify-center gap-2 px-6 h-12"

アイコンは必ず中央。isIconOnly の場合は正円を維持。

Dropdown: radius="md" (12px), content: "rounded-[16px] shadow-2xl"

Link: className="text-primary font-medium hover:underline"

Navbar: className="h-16 px-8 bg-background/80 backdrop-blur-md fixed top-0 w-full"

Pagination: radius="full" cursor: "bg-primary text-white"

Tabs: variant="solid" radius="full" classNames={{ tabList: "bg-content2/50 p-1", cursor: "bg-primary shadow-lg" }}

3.2 フォーム・入力系 (Forms)

Input / Textarea / Number Input: variant="flat" radius="lg" (16px)

classNames={{ inputWrapper: "bg-content2/40 px-4 h-14 focus-within:bg-background border-b-2 border-transparent focus-within:border-primary shadow-inner transition-all" }}

Autocomplete / Select: popoverProps={{ className: "rounded-[20px] shadow-2xl" }}

Checkbox / Radio Group / Checkbox Group: color="primary" label: "ml-2 font-medium"

Date Input / Picker / Range Calendar / Calendar: radius="lg" calendarProps={{ className: "rounded-[28px] bg-background p-4 shadow-xl" }}

Input OTP: radius="md" input: "border-2 border-transparent focus:border-primary h-14"

Slider: color="primary" classNames={{ track: "bg-content2", thumb: "bg-primary shadow-lg border-2 border-background" }}

Switch: radius="full" classNames={{ wrapper: "group-data-[selected=true]:bg-primary shadow-none" }}

Time Input: variant="flat" radius="lg"

3.3 データ表示 (Data Display)

Avatar / User: radius="lg" (16px) / name: "font-bold text-on-surface"

Badge / Chip: radius="full" variant="flat" className="bg-primary-50 text-primary-900 px-3"

Card: (2.A参照) rounded-[28px] bg-content1 p-8 shadow-none w-full

Code / Snippet: className="bg-content2 rounded-lg text-primary"

Divider: className="bg-divider/30 my-6"

Image: radius="lg" (20px) shadow-none

Listbox: item: "rounded-full px-6 py-2 my-1"

Scroll Shadow: className="max-h-[400px]"

Table: radius="lg" (20px) th: "bg-content2/50 text-primary font-bold py-4"

3.4 フィードバック & ユーティリティ (Feedback & Utility)

Alert: variant="flat" radius="lg" className="p-4 gap-4" (文字色は1.C参照)

Circular Progress / Progress: color="primary" classNames={{ track: "bg-content2", indicator: "bg-primary" }}

Drawer / Modal: radius="lg" (28px) base: "bg-background/90 backdrop-blur-xl"

Kbd: className="bg-content2/50 border-none font-mono text-primary"

Popover / Tooltip: radius="md" (12px) className="bg-content2 shadow-xl border-none"

Skeleton: radius="lg" (16px) before:bg-primary-50 (200ms以上のロードに必須)

Spacer: 原則 gap クラスを使い、緊急時のみ使用せよ。

Spinner: color="primary" size="sm"

Toast: radius="full" className="bg-on-surface text-background font-bold px-8 shadow-2xl"

4. 品質保証チェックリスト (Final Guardrails)

❌ 可変幅コンテナ禁止: カードの幅が文字数に左右されていないか？(w-full等の固定幅を使用しているか)

❌ アイコン整列不備: ボタン内アイコンは flex items-center justify-center で中心にあるか？

❌ 純白 (#FFFFFF) 禁止: 全ての背景に #fff8f6 を徹底しているか？

❌ 警告・エラーの文字色不備: 定義された On-Container カラーを使用しているか？

✅ Skeletonの多用: 非同期データ読み込みには必ずスケルトンを配置しているか？
